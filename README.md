# Would you rather?

This is a minimalistic app that allows you to login as a user, post questions,
answer questions, and check the leaderboard.

## Instructions
##### Develop
1. Run `npm install`
2. Run `npm start`

##### Build
1. Run `npm install`
2. Run `npm build`

## Notes
The app currently uses a mock back-end which can be swapped out.

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
